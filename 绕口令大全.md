# 🗣️ 绕口令大全

> 每天练习10分钟，嘴皮越来越溜

---

## 🌱 入门级

### 1. 八百标兵
```
八百标兵奔北坡，北坡八百炮兵炮。
标兵怕碰炮兵炮，炮兵怕把标兵碰。
```

### 2. 调到敌岛
```
调到敌岛打特盗，特盗太刁投短刀。
挡推顶打短刀掉，踏盗得刀盗打倒。
```

### 3. 哥跨瓜筐
```
哥跨瓜筐过宽沟，快跨瓜筐沟宽走。
跨沟跨瓜跨瓜走，瓜筐挂沟沟绊瓜。
```

### 4. 牛郎和刘娘
```
牛郎恋刘娘，刘娘念牛郎。
牛郎年年恋刘娘，刘娘年年念牛郎。
郎恋娘来娘念郎。
```

### 5. 老农和老龙
```
老龙和老农，
老农要喝老龙的老龙井，
老龙要供老农的老龙头。
农说井里龙浸水，
龙说井上龙吸农。
```

## 🌿 中等级

### 6. 六十六岁刘老六
```
六十六岁刘老六，推着六十六只大油篓，
六十六枝垂杨柳，扎着六十六个马马蔸。
```

### 7. 扁担长
```
扁担长，板凳宽，扁担没有板凳宽，板凳没有扁担长。
扁担绑在板凳上，板凳不让扁担绑在板凳上。
```

### 8. 化肥会挥发
```
化肥会挥发，
黑灰会挥发，
化肥挥发会黑灰，
黑灰挥发会化肥。
```

### 9. 喇嘛和哑巴
```
打南边来了个喇嘛，手里提拉着五斤鳎犸。
打北边来了个哑巴，腰里别着个喇叭。
南边提拉鳎犸的喇嘛要拿鳎犸换北边别喇叭哑巴的喇叭。
哑巴不愿意拿喇叭换喇嘛的鳎犸，
喇嘛非要换别喇叭哑巴的喇叭。
```

### 10. 墙头有根藤
```
墙头有根藤，藤上挂个铃。
风儿吹动藤，藤儿摇动铃。
铃儿响叮当，铃儿响叮当。
```

## 🌳 进阶级

### 11. 报菜名
```
蒸羊羔、蒸熊掌、蒸鹿尾儿、烧花鸭、烧雏鸡、烧子鹅，
卤猪、卤鸭、酱鸡、腊肉、松花、小肚儿、晾肉、香肠儿。
```

### 12. 红鲤鱼
```
红鲤鱼家有头红驴鱼，红驴鱼要借红驴鱼的红驴。
红鲤鱼不借红驴鱼的红驴，气得红鲤鱼变红驴。
```

### 13. 白石塔
```
白石白又滑，搬来白石搭白塔。
白石塔，白石塔，白石搭石塔，白塔白石搭。
搭好白石塔，白塔白又滑。
```

### 14. 吃葡萄不吐葡萄皮
```
吃葡萄不吐葡萄皮，
不吃葡萄倒吐葡萄皮。
```

### 15. 四是四，十是十
```
四是四，十是十，十四是十四，四十是四十。
谁把四十说十四，谁的舌头没仔细。
谁把十四说四十，谁的舌头吃了大亏。

## 🏆 高阶级

### 16.  UNIT音韵练习
```
调到敌岛打特盗，特盗太刁投短刀。
挡推顶打短刀掉，踏盗得刀盗打倒。

七加一、七减一，加完减完等于几？
七加一、七减一，加完减完还是七。

八百标兵奔北坡，炮兵并排北边跑。
炮兵怕把标兵碰，标兵怕碰炮兵炮。

### 17.  综合绕嘴
```
出东门，过大桥，大桥前面一树枣，
拿着竿子打枣，打，青的，红的，都好，
拿到家里妈妈吃，爸爸吃，爷爷吃，奶奶吃，
就是不让小涛吃，气的涛哥哇哇哭。
```

### 18.  玲珑塔
```
玲珑塔，塔玲珑，玲珑宝塔十三层。
塔前有座庙，庙里有老僧，老僧当方丈，
小僧当和尚。
```

## 📅 练习建议

| 难度 | 每天练习 | 时间 |
|------|---------|------|
| 入门 | 3-5个 | 5分钟 |
| 中级 | 2-3个 | 10分钟 |
| 进阶 | 1-2个 | 15分钟 |
| 高级 | 1个 | 20分钟 |

**技巧：**
1. 先慢读，读准每个字
2. 逐步加快速度
3. 录音回听，找问题
4. 对着镜子练，观察口型
